How to check if Object is empty?

```js
/**
 * How to check if Object is empty?
 */

function isEmpty(obj) {
  // if an object has properties the loop would be runed
  for (let key in obj) return false;
  return true;
}

isEmpty({}); // true
isEmpty({ x: 1 }); // false
```
